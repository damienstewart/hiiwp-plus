HiiWP Plus  - WordPress plugin

== Description ==

Add a portfolio to your HiiWP Theme.

Looking for a helping hand? [View plugin documentation](https://hiilite.ticksy.com/).

== Usage ==

This plugin is an add-on for the HiiWP Theme. You can download the HiiWP Theme from ThemeForest.

== Installation ==

Installing "HiiWP Plus" can be done either by searching for "HiiWP Plus" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org.
1. Upload the ZIP file through the "Plugins > Add New > Upload" screen in your WordPress dashboard.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit the settings screen and configure, as desired.
